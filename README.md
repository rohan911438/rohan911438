I build production-minded Web3 products with a bias toward agentic automation, cross-chain payments, on-chain escrow, and developer-first infrastructure. The goal is simple: ship systems that are useful, auditable, and easy to demo.

## Focus

- AI agents that negotiate, transact, and execute on-chain.
- DeFi products with real UX, not just smart-contract proofs.
- Protocol tooling that lowers developer friction and increases adoption.
- Open-source projects that can survive hackathons and evolve into real products.

## Featured Projects

### [A2A Protocol](https://github.com/rohan911438/A2A-Protocol)
Autonomous agents that negotiate deals, gate actions with payment authorization, and settle through Stellar escrow. This is my strongest signal for the AI-agent economy: clear architecture, real deployment, and a full end-to-end workflow.

Tech stack: React, FastAPI, LLM negotiation, Stellar Soroban, x402-style payment gating.

Links: [Live Frontend](https://a2aprotocol.netlify.app/) • [Backend](https://a2a-protocol-rn62.onrender.com) • [Demo](https://youtu.be/3KrVJvXWhu8?si=Ixmrt1isNr5Y-pZn)

### [StreamPay](https://github.com/rohan911438/Streampay)
Privacy-first cross-chain subscriptions on Solana. Strong real-world usefulness here: merchant UX, SDK distribution, and a clear value proposition around confidential recurring payments.

Tech stack: Next.js, Solana, LI.FI, Jupiter, Cloak, TypeScript, SDK-first architecture.

Links: [Live App](https://streampay-web.vercel.app) • [SDK](https://www.npmjs.com/package/streampay-sdk) • [Demo](https://youtu.be/L0BmvmVClk4?si=8zuvj-owzLILCGVa)

### [ArbitPy](https://github.com/rohan911438/ArbitPy)
Python-to-blockchain compiler tooling for Arbitrum Stylus. This stands out for technical depth: it turns developer experience into a product and speaks directly to protocol infrastructure and compiler innovation.

Tech stack: Python compilation pipeline, Arbitrum Stylus, Solidity, TypeScript SDK, AI-assisted tooling.

Links: [Live Demo](https://arbitpy-ebon.vercel.app/) • [NPM SDK](https://www.npmjs.com/package/arbitpy-sdk) • [Demo](https://youtu.be/FVWdfZuF8ps?si=d_2pfwkMBGAvLFDI)

### [IP-Fi Explorer](https://github.com/rohan911438/ip-fi-explorer)
Fractional IP investment and royalty automation on Story Protocol, plus a reusable widget package. It is one of the best examples of product thinking in the portfolio because it combines ecosystem utility with developer distribution.

Tech stack: React, Vite, Node.js, Express, MongoDB, Story Protocol, TypeScript widget library.

Links: [Live App](https://ip-fi-explorerr-dgnl.vercel.app/) • [Widget NPM](https://www.npmjs.com/package/ip-fi-swap-widget) • [Demo](https://youtu.be/VWqBWKWBwzk?si=BD-Kyc0Xs2D9nxsE)
